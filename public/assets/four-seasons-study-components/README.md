# Four Seasons Study Component Assets

These PNG assets were prepared from the selected "清供书斋" visual direction for both mobile portrait and web landscape layouts.

## Transparent Components

- `almanac-plaque-blank.png` - blank hanging almanac plaque for calendar/date content.
- `solar-term-flower-branch.png` - seasonal plant object for solar-term changes.
- `wuxing-clothing-chest.png` - five-color clothing guide object.
- `seasonal-produce-basket.png` - seasonal produce basket.
- `ancient-guest-scholar.png` - daily ancient guest character layer.
- `visitor-name-plaque-blank.png` - blank visitor name plaque for dynamic guest text.
- `letter-archive-cabinet.png` - letter/archive cabinet for chat history state.

## Background Layer

- `window-landscape-background.png` - separate rectangular scenery layer for the window. Place weather particles between the window frame and this background.

## Fixed Layout Layers

- `portrait-fixed-background-room.png` - portrait rear room layer behind the character, including hollow window frame, rear shelves, plants, and fixed furnishings.
- `portrait-fixed-foreground-desk.png` - portrait foreground layer in front of the character, including the fixed desk and stationery objects.
- `landscape-fixed-background-room.png` - landscape rear room layer behind the character, including hollow window frame, rear shelves, plants, and fixed furnishings.
- `landscape-fixed-foreground-desk.png` - landscape foreground layer in front of the character, including the fixed desk and stationery objects.

Suggested layer order, back to front:

1. Window landscape background
2. Weather/season particles
3. Fixed background room layer
4. Ancient guest character
5. Dynamic modules such as almanac, solar-term plant, clothing chest, produce basket, visitor plaque, and archive cabinet
6. Fixed foreground desk layer

## Preview

- `preview-contact-sheet.png` - quick preview sheet only; not intended as an app asset.
- `fixed-layers-preview.png` - quick preview sheet for portrait/landscape fixed layers only.

All component assets are PNG. Text has been removed from the generated component assets so app text can be added dynamically.
